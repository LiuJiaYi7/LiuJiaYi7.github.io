Linux基本命令使用方法

基本命令
发布日期:   2021-05-04

## 一、基本命令

###1、关机与重启

关机
shutdown -h now 立刻关机
shutdown -h n 多长时间后关机（n 表示输入想要关机的时间数）
例如： shutdown -h 5 五分钟后关机
power off 立刻关机
重启
shutdown -r now 立刻重启
shutdown -r n 多长时间后重启
例如： shutdown -r 5 五分钟后重启
reboot 立刻重启

###2、帮助命令

help 命令
shutdown --help
ifconfig --help 查看网卡信息
man 命令（命令说明书）
man shutdown
注意：想要退出命令说明书，点击 q 键即可！

###3、查看历史命令

命令：history

## 二、目录操作命令

###1、目录切换命令

命令：cd
cd / 切换到根目录
cd /user 切换到根目录下的 user 目录
cd ../ 切换到上一级目录
cd ./ 表示该目录本身
cd ~ 切换到 home 目录
cd - 切换到上次访问的目录

###2、目录内容显示

命令：ls、pwd
ls 查看当前目录下的所有目录和文件
ls -l 查看当前目录下的所有目录和文件（列表查看，显示更多详细信息）
ls -a 查看当前目录下所有目录和文件（包括隐藏的文件）

ls -c 按文件修改时间排序并显示
ls -i 在输出的第一行显示文件的索引节点号并显示
ls -r 按逆序显示 ls 命令的输出结果
ls -R 递归地显示指定目录的各个子目录中的文件
ls /dir 查看指定目录下的所有目录和文件
例如： ls /um 查看 um 目录下所有目录和文件
pwd 显示当前工作目录的绝对途径，无任何选项或参数

###3、目录操作命令（包括对目录的增、删、改、查）

####3-1、创建目录（增）

命令：mkdir
mkdir 目录名 在当前目录下创建一个名为 XXX 的目录
例如： mkdir abc 在当前目录下创建一个名为 abc 的目录
mkdir /user/目录名 在指定目录下创建一个名为 XXX 的目录
例如： mkdir /user/abc 在 user 目录下创建一个名为 abc 的目录
注意：若无管理员权限，则在命令前加上 sudo 或者 su 命令

####3-2、删除目录或文件（删）

命令：rm/rmkdir
删除文件:
rm 文件名 删除当前目录下的 XXX 文件
rm -f 文件名 删除当前目录下的 XXX 文件（不予询问）
删除目录：
rm -r 目录名 递归删除当前目录下的 XXX 目录（-r 表示递归）
rm -rf 目录名 递归删除当前目录下的 XXX 目录（不与询问）
rmdir 目录名 删除当前目录下的一个或多个空的子目录
rmdir -p 目录名 递归删除当前目录下的 XXX 目录（-p 表示递归）
注意：若无管理员权限，则在命令前加上 sudo 或者 su 命令

####3-3、目录修改（改）

命令：mv、cp
目录重命名：
mv 当前目录名 新目录名 把当前目录名改为新目录名
例如： mv abc aaa 将目录 abc 改为 aaa
注意：mv 命令同样可以对文件和压缩包等进行重命名操作，不只局限于目录！
目录拷贝：
cp -r 目录名称 目录将要拷贝的位置 将当前目录下的目录拷贝到目标位置
例如： cp -r /home/abc /um 将 home 目录下的 abc 目录拷贝到 um 目录下
注意：cp 拷贝命令对压缩包和文件同样有效，但拷贝文件和压缩包时不用递归命令
目录剪切：
mv 目录名称 目录目标位置 将当前目录下的 XXXX 目录剪切到目标位置
例如： mv /home/abc /um 将 home 目录下的 abc 目录剪切到 um 目录下

####3-4、目录搜索（查）

命令：find
find 目录 参数 文件名称
例如： find /home/tmp -name ‘a‘ 查找该目录下所有以 a 开头的目录或文件

## 三、文件操作命令

文件操作（增，删，改，查）

###1、新建文件（增）

命令：touch
touch 文件名
例如：  touch  a.txt 在当前目录创建一个名为 a.txt 的文件

###2、删除文件 （删）

命令：rm
rm -rf 文件名
例如： rm -rf a.tx 删除当前目录下一个名为 a.txt 的文件

###3、修改文件（改）

编辑器：vi、vim

####3-1、打开文件：

命令： vi 文件名
示例：打开当前目录下的 aa.txt 文件     vi aa.txt 或者 vim aa.txt
注意：使用 vi 编辑器打开文件后，并不能编辑，因为此时处于命令模式，点击键盘 i/a/o 进入 编辑模式。

####3-2、编辑文件：

使用 vi 编辑器打开文件后点击按键：i ，a 或者 o 即可进入编辑模式。
i: 在光标所在字符前开始插入
a: 在光标所在字符后开始插入
o: 在光标所在行的下面另起一新行插入

####3-3、保存文件：

第一步：ESC  进入命令行模式
第二步: :   进入底行模式
第三步：wq     保存并退出编辑

####3-4、取消编辑：

第一步：ESC  进入命令行模式
第二步：:     进入底行模式
第三步：q!     撤销本次修改并退出编辑

###4、查看文件（查）

命令：cat/head/more/tail/less/od
cat 文件名 查看文件最后一屏内容
head 文件名 在屏幕上显示文件的开头若干行或多少个字节
head -n 文件名 指定从文件开头的显示行数，默认为十行
more 文件名 查看文件内容（可逐页显示）
less 文件名 分页显示文件内容
tail 文件名 查看文件后面的内容
tail -n 文件名 查看文件后几行的内容
od 文件名 将指定文件以八进制形式转出到标准输出

命令：sort
sort 文件名 对 XXX 文件内容按字母出现的先后顺序排列

###6、比较文件内容

命令：comm/diff
comm 文件名 另一个文件名 对两个已经排好序的文件内容进行比较
例如： comm a.txt b.txt 对 a.txt 和 b.txt 的文件内容进行比较
diff 文件名 另一个文件名 比较两个文本文件并找出他们的不同，根据不同的行列内容进行修改是他们内容相同

###7、文件内容迁移

命令：mv
mv 文件名 另一个文件名 将前一个文件的内容转移到后一个文件
mv a.txt b.txt 将 a 文件的内容转移的 b 文件中
cat 文件名 另一个文件名 >新的文件名 将两个文件的内容合并起来转存到另一个新的文件中
例如： cat f1 f2>f3 将 f1f2 两个文件的内容转存到文件 f3 中

###8、文件内容查找

命令：grep
grep 选项 模式 文件 Minh 在文本文件中查找指定模式的单词或词语并在标准输出上显示包括给定字符串模式的所有行
例如： grep -i 'home' /etc/passwd
-i 表示忽略大小写
- x 表示强制整行匹配
- w 表示强制关键字完全匹配
- e 表示定义正则表达式
- m 定义多少次匹配后停止搜索
- n 指定输出的同时打印行号
- H 为每一匹配项打印文件名
- r 在指定目录中进行递归查询

###9、文件内容统计

命令：wc
wc 文件名列表 统计出指定文件的字节数、字数、行数并输出结果
输出格式：行数 字数 字节数 文件名
wc -c 文件名 统计字节数
wc -l 文件名 统计行数
wc -w 文件名 统计字数

###10、文件查找

命令：find、locate
find 文件路径 匹配表达式 -exec 外部命令 {} \; 在目录结构中搜索满足查询条件的文件并执行指定操作（没有管理员权限在命令行前面加 sudo）

例如:sudo find /etc -name “passwd” -exec grep “wang” {} ; 首先匹配到所有文件名为 “passwd” 的文件，再查找这些文件中含有 “wang” 用户的文件

locate [选项] [模式] 查找功能更强大，直接在数据库中查找，无需使用 sudo 命令

例如： locate chgpasswd 查找所有含有 chgpasswd 的文件路径
- d 指定 locate 命令所使用的数据库
- c 只列出查到的条目数量
- A 列出匹配的所有条目
- w 匹配整个路径

###11、创建链接文件

命令：ln
ln -s 文件或目录名 链接文件名 在文件之间创建链接
ln 原文件 链接文件 建立链接文件

###12、文件压缩与解压缩、打包与解包

命令：gzip、unzip、tar
文件压缩：
gzip 文件名 压缩当前目录下的 XXX 文件
tar zcvf 压缩文件名.tar.gz 文件名 单个文件压缩打包
tar zcvf 压缩文件名.tar.gz 文件1 文件2... 多个文件压缩打包
文件解压缩：
gunzip 解压缩文件名
gzip -d 解压缩文件名
tar zxvf 解压缩文件名.tar.gz

###13、权限修改

rwx：r 代表可读，w 代表可写，x 代表该文件是一个可执行文件，如果 rwx 任意位置变为 - 则代表不可读或不可写或不可执行文件。
第一段（3 位）：代表拥有者的权限
第二段（3 位）：代表拥有者所在的组，组员的权限
第三段（最后 3 位）：代表的是其他用户的权限
 例如：421  421  421

  • rw-   —     —
    命令：chmod
    chmod 选项 文件名
    例如： chmod u+r aaa.txt 将 aaa 文件的所有者权限加上可读
    chmod u-r aaa.txt 将 aaa 文件的所有者权限删去可读
    或者采用 8421 法
    例如： chmod 377 aaa.txt 将 aaa 文件的所有者权限删去可读
    注意：若无管理员权限，则在命令前加上 sudo 或者 su 命令

###14、变更文件访问者身份

####14-1、变更所有者

命令：chown
chown 选项 新所有者 文件列表 变更文件所有者，是其他用户对文件具有使用权

-R 表示递归变更，目录连同其子目录下的所有文件的所有者都变更

注意：执行chown命令需要root权限，在命令行前面加上sudo

例如： sudo chown wang news 将 news 的所有者改为 wang

####14-2、变更所属组

命令：chgrp
chgrp 选项 新的所属组名 文件列表

-R 表示递归变更，目录连同其子目录下的所有文件的所有者都变更所属组

chown 选项 新所有者 ：新所属组 文件列表 同时变更文件的所有者和所属组 `

注意：执行chgrp命令需要root权限，在命令行前面加上sudo

## 四、管理用户和组

###1、管理用户账户

命令：grep
cat /etc/passwd | grep 用户名 通过查看用户配置文件来查找其中包含该用户名的文件

###2、添加用户账户

命令：useradd、adduser
useradd -选项 用户名 添加用户
- d 用于指定用户主目录
- g 用于指定该用户的所属组
- G 用于指定用户所属其他组列表
- r 用于创建一个系统账户（不会创建主目录，默认为 /bin/bash）
-s 指定用户登录时使用的 shell 脚本
- u 指定新用户的 UID
注意：无root使用权限时，在命令行前面加sudo即可
- D 用于显示或更改默认的 useradd 配置
useradd -D 选项 用于修改默认的 useradd 配置
- g 表示默认组
- b 表示默认目录
- f 表示过期禁用
- e 表示过期日期
- s 表示默认 shell
sudo adduser 用户名 添加用户并设置相关信息

###3、管理用户账户密码

passwd 选项 用户名 为用户设置密码
例如： passwd wang 为用户 wang 设置密码
passwd -l 用户名 锁定账户密码
passwd -d 用户名 删除账户密码
passwd -S 用户名 查看某账户当前的状态
passwd -u 用户名 解除账户密码锁定
注意：无root使用权限时，在命令行前面加sudo即可

###4、修改用户账户

命令：usermod、chfn
usermod 选项 用户名 修改用户的各项信息

usermod -l 新用户名 原用户名 更改用于账户名

usermod -L 用户名 锁定该用户账户，临时禁止登录

usermod -U 用户名 解除锁定

chfn 选项 用户名 更改用户的个人信息

chfn -f 用户名 更改用户的全名（真实姓名）

chfn -h 用户名 更改用户的家庭电话

chfn -o 用户名 更改用户的办公地址

注意：无root使用权限时，在命令行前面加sudo即可

###5、删除用户账户

命令：userdel、deluser
userdel 选项 用户名 删除用户账户

userdel -r 用户名 递归删除该账户下的主目录和邮件目录

注意：不允许删除正在使用的用户账户 `

deluser --remove-home 用户名 表示同时删除用户的主目录和邮箱

deluser --remove-all-files 用户名 表示删除用户拥有的所有文件

deluser --backup 用户名 表示删除用户之前将文件备份

deluser --system 用户名 表示只有当该用户是系统用户时才删除

deluser --backup-to 目录名 用户名 表示指定备份的目标目录

###6、创建组账户

命令： groupadd
groupadd 选项 组名 创建一个组账户

groupadd -g 组名 自行指定组的 GID

groupadd -r 组名 创建系统组，GID 值小于 500（不带此选项，则创建的为普通组
）
addgroup --gid ID 组名 添加一个普通用户组

addgroup --system --gid 组 ID 组名 添加一个管理员用户组

###7、修改组账户

命令： groupmod -g GID -n 新组名 组名 修改用户组的组名和 GID 值

###8、删除组账户

命令：groupdel 、delgroup
groupdel 组名 删除用户组

delgroup ---system 组名 表示只有当该用户组是系统用户组时才删除

delgroup --only-if-empty 表示只有当该用户组中无成员时才删除

###9、管理组成员

####9-1、显示某用户所属的全部组

命令：groups
注意：如果没有指定用户名则默认为当前登录用户

####9-2、将用户添加到指定的组

命令：gpasswd、adduser
gpapsswd -a 用户名 组名 将用户添加到指定的组，使其成为该组的成员

gpapsswd -d 用户名 组名 将用户从该组中删除

gpasswd -M 用户名，用户名，... 组名 将多个用户添加到该组中，使之成为组成员

adduser 用户名 组名 将用户添加到指定的组，使其成为该组的成员

deluser 用户名 组名 将用户从该组中删除

###10、查看其它用户管理命令

####10-1、查看用户信息

命令：id
id 选项 用户名 查看指定用户或当前用户的信息
注意：若不指定用户名，则将显示当前登录的用户的信息，若指定用户名，则显示该账户信息

####10-2、查看登录用户

命令：who 、last、w
who 显示当前登陆的用户账户
last 查看系统的历史登陆情况
last 用户名 查看某个用户的历史登陆情况
last -n 查看系统最近几次的登陆情况
w 查看用户的执行进程

## 五、编译 C 程序

###1、安装 gcc 编译器

命令： sudo apt install gcc
注意：安装之后可以用该命令查看版本:gcc -v

###2、开始编译

命令： gcc 选项 源文件 注意：保存源文件时将拓展名设置为文件名.c`

sudo gcc -o 源文件名称 对该源文件进行预处理、编译、汇编并连接形成可执行文件

-E 对源文件进行预处理，生成名为文件名.i的源代码文件

-S 对源文件进行预处理和编译，生成名为文件名.s的汇编代码文件

-c 对源文件进行预处理、编译和汇编，生成名为.c同名目标文件

注意：若无指定输出文件，则默认输出文件名为 a.out

###3、在命令行中执行程序

命令： ./文件名 完成编译和连接后在命令行中执行
