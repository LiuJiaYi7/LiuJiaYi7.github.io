# 搭建 hexo 时踩过的坑及解决办法（GitHub 托管）

## 一、hexo 简介

Hexo 是一个简单地、轻量地、基于 Node.js 的一个静态博客框架，可以方便的生成静态网页。

# 二、安装 hexo

# 1、安装 hexo 前提

安装 Hexo 相当简单，只需要先安装下列应用程序即可：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

####1.1、安装 Git

Windows：下载并安装 git.(注意：该软件最好安装在除 C 盘外的其他盘，点击下载好的 exe 文件，一路 next 即可安装完成)
Mac：使用 Homebrew, MacPorts 或者下载 安装程序。

在 git bash 命令行窗口中输入以下命令即可查看下载版本

####1.2、安装 Node.js

Node.js 为大多数平台提供了官方的 安装程序。对于中国大陆地区用户，可以前往 淘宝 Node.js 镜像 下载。当然，对于玩电脑很溜，接触过外网或者有 VPN 及其他的翻墙软件也是可以在官网上下载的，速度也很快。（注意事项与安装 git 相同）
其它的安装方法：
Windows：通过 nvs（推荐）或者 nvm 安装。
Mac：使用 Homebrew 或 MacPorts 安装。
Linux（DEB/RPM-based）：从 NodeSource 安装。
其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的 指导
在电脑自带的 cmd 窗口输入命令： node 可查看 Node.js 的版本

# 1、创建一个 GitHub 账号并登录

没有接触过 GitHub 的同学可以看一下，接触过的大佬可以自动忽略啊～～～

浏览器搜索 GitHub 官网并打开，在首页输入自己的邮箱，点击 sign up for github

之后进入该页面，创建个人账户，输入自己的用户名（英文），邮箱地址和用户密码，验证你的账户后点击 create account 即可，到此时，你的个人 GitHub 账户已经创建成功了！

# 2、创建一个 repo（仓库）

名称最好命名为 ```yourname.github.io`, 其中 yourname 是你的账户名
